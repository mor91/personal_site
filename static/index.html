    Redditor-Bot with Seq2seq

    In this project I used the sequence to sequence framework to model reddit conversations created from the comments sections.

    I trained an attention based seq2seq model on roughly 20m exchanges between users.

    I prepared the reddit dataset using Google BigQuery.

    Visualizing word embeddings

    In this project I trained a recurrent neural network language model on the Penn Tree Bank dataset. A language model is a model that tries to predict a word given the preceding words.

    After the training is done I visualized the created word representations using Tensorflow new embedding visualizer tool with t-SNE.

    The model is a Long Short-Term Memory network

    IMAGE GENERATION

    Using the recently proposed Generative Adversarial Networks framework I generated MNIST images. The GAN framework consists of a generator and discriminator, the generator generates images from random vectors and the discriminator tried to discriminate between real samples and generated samples.

    Also, I compared convolutional architectures to fully-connected and linear ones.
